course-site-manager
===================

```
Usage: csm [options] [command]

  Commands:

    list [options] [key]           Lists all installed instances.
    pull [options]                 Pull in and build the master course-site.
    update <key>                   Update a course-site instance with the master instance.
    install <name> <dir> [webdir]  Install a new instance.

  Options:

    -h, --help     output usage information
    -V, --version  output the version number

```
